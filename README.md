# Remote Project Manager (Home Assistant Add-on)

Manage remote Docker Compose projects over SSH from Home Assistant.

## About

This add-on packages the Remote Project Manager FastAPI service and web UI. The UI lets you register SSH hosts, manage compose projects, view logs, and trigger updates/backups.

Full add-on documentation: see `DOCS.md`.

Builds pull the upstream release tarball defined by `RPM_VERSION` in `build.yaml`.
